# 🏋️‍♂️ Workout Tracker App

A dynamic frontend web application designed to help users log, track, and visualize their daily fitness routines and workout sessions.

## 🎯 The Problem Solved
Tracking workouts using pen and paper or basic notes apps can be cumbersome and lacks visual motivation. This application provides a streamlined, interactive interface for fitness enthusiasts to easily input exercises, track their progress over time, and stay motivated.

## 🛠️ Tools & Technologies
*   **Framework:** React (Vite)
*   **Language:** TypeScript
*   **Styling:** Custom CSS
*   **Architecture:** Component-Based UI

## ✨ Key Features & Techniques
*   **Interactive UI:** Clean and intuitive interface for logging sets, reps, and weights.
*   **State Management:** Efficient handling of dynamic workout data using React hooks.
*   **Component Modularity:** Reusable UI components for consistent design and easier maintenance.
*   **Cross-Device Compatibility:** Responsive design ensuring the app is usable at the gym on a phone or at home on a desktop.

## 🚀 How to Run Locally
1. Clone the repository: `git clone <https://github.com/SamJonathan06/workout-trackerrr.git>`
2. Navigate to the directory: `cd workout-tracker`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
